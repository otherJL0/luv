version ()
Returns the libuv version packed into a single integer. 8 bits are used for each component, with the patch number stored in the 8 least significant bits

Returns:

    integer version

Usage:

    uv.version() -- This would be 0x010203 in libuv 1.2.3
version_string ()
Returns the libuv version number as a string. For non-release versions, the version suffix is included.

Returns:

    string version

Usage:

    uv.version_string() -- This would be "1.2.3" in libuv 1.2.3
